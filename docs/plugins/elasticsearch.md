# Plugin elasticsearch
## Variables
| Variable name | Default | Applied function |
|---|---|---|
| pattern | - |  |
| logLevel | info |  |
| host | - |  |
| port | - |  |
| scheme | scheme |  |
| sslVerify | true |  |

## Plugin template
```
<match {{ .pattern }}.**>
  @type elasticsearch
  @log_level {{ .logLevel }}
  include_tag_key true
  type_name fluentd
  host {{ .host }}
  port {{ .port }}
  scheme  {{ .scheme }}
  {{- if .sslVerify }}
  ssl_verify {{ .sslVerify }}
  {{- end}}
  {{- if .sslVersion }}
  ssl_version {{ .sslVersion }}
  {{- end}}
  logstash_format {{ .logstashFormat }}
  logstash_prefix {{ .logstashPrefix }}
  reconnect_on_error true
  {{- if .user }}
  user {{ .user }}
  {{- end}}
  {{- if .password }}
  password {{ .password }}
  {{- end}}
  log_es_400_reason {{ .log_es_400_reason }}
  <buffer tag, time>
    @type file
    path {{ .bufferPath }}
    timekey {{ .timekey }}
    timekey_wait {{ .timekey_wait }}
    timekey_use_utc {{ .timekey_use_utc }}
    flush_mode interval
    retry_type exponential_backoff
    flush_thread_count {{ .flush_thread_count }}
    flush_interval {{ .flush_interval }}
    retry_forever {{ .retry_forever }}
    retry_max_interval {{ .retry_max_interval }}
    chunk_limit_size {{ .chunkLimit }}
    queue_limit_length {{ .queueLimit }}
    overflow_action block
  </buffer>
</match>
```
